# ✨ GlowAI Chennai

## AI-Powered Beauty Salon Marketplace

GlowAI Chennai is a modern AI-powered beauty salon marketplace designed to help users discover trusted salons, receive personalized beauty recommendations, compare services, and book appointments seamlessly.

Built as a startup-grade platform with AI-driven beauty assistance, salon discovery, and appointment management.

---

## 🚀 Problem Statement

Finding reliable beauty salons, personalized beauty guidance, and budget-friendly beauty services is often difficult for customers.

Users struggle with:

- Identifying trustworthy salons
- Choosing suitable beauty treatments
- Comparing salon services and pricing
- Getting personalized beauty recommendations
- Finding affordable salons within their budget

---

## 💡 Solution

GlowAI Chennai combines Artificial Intelligence and Beauty Marketplace features into a single platform.

The platform helps users:

- Discover top-rated salons
- Get AI-powered beauty consultations
- Receive personalized recommendations
- Compare beauty services
- Book appointments online
- Explore budget-friendly and women-owned salons

---

## 🌟 Key Features

### 🤖 AI Beauty Advisor
Provides beauty consultation and recommendations based on user concerns.

Examples:

- Hair fall solutions
- Curly hair care
- Skincare routines
- Bridal skincare preparation

### 🎯 AI Salon Matchmaker

Users describe their needs in natural language.

Examples:

- Bridal makeup under ₹5000
- Best salon near T Nagar

- Hair spa for damaged hair

The AI recommends suitable salons.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/glowai-chennai.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file inside the backend directory.

```env
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
JWT_EXPIRES_IN=7d
GEMINI_API_KEY=your_gemini_api_key
PORT=5000
NODE_ENV=development
FRONTEND_URL=http://localhost:5173
```
